
# Music and Mental Health Project

## Описание проекта
Этот проект посвящён анализу данных о музыкальных привычках людей и их психологическом состоянии.

В качестве данных используется таблица с результатами опроса, где есть информация о:
- возрасте
- количестве часов прослушивания музыки
- любимом жанре
- музыкальном сервисе
- влиянии музыки на состояние
- уровнях тревожности, депрессии, бессонницы и OCD

Цель проекта — показать, какие выводы можно сделать на основе реальных данных.

## Цель проекта
Создать консольное Python-приложение для анализа данных с использованием:
- pandas
- numpy
- генераторов
- декораторов
- CLI
- тестов pytest

## Что реализовано по заданию
1. pandas
Используется для:
загрузки данных
очистки данных
фильтрации
группировки
агрегирующих функций

2. numpy
Используется для:
вычисления среднего значения
вычисления медианы
вычисления стандартного отклонения

3. Генераторы

В проекте реализованы генераторы:
insight_generator() — выводит аналитические выводы по одному
report_generator() — формирует строки отчёта по одной

4. Декораторы
В проекте реализованы декораторы:
log_call — логирует вызов функции
measure_time — измеряет время выполнения функции

5. CLI

Проект запускается через терминал командами:

python main.py stats
python main.py analyze
python main.py report
python main.py genres
python main.py services
